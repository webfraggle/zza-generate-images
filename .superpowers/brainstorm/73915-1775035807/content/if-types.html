Drei Arten von if-Bedingungen

Wie werden sie im Node-Editor unterschieden?

① Layer-Sichtbarkeit

YAML: if: direkt auf dem Layer

- type: rect
  if: "greaterThan(zug1.abw, 0)"
  x: 0
  y: 60
  color: "#FF0000"

→ Steuert ob der gesamte Node gerendert wird. Sichtbar als Badge oben rechts am Node.

② Bedingte Eigenschaft

YAML: if/then/else auf einem Feld

color:
  if: "greaterThan(zug1.abw, 0)"
  then: "#FF4444"
  else: "#FFFFFF"

→ Steuert den Wert eines einzelnen Feldes. Sichtbar als kompakte wenn/dann/sonst-Chips direkt neben dem Feld.

③ Block-Bedingung

YAML: Block-Node ohne type:

- if: "startsWith(nr,'ICN')"
  layers:
    - type: image
      file: icn.png
    - type: text
      value: "ICN Express"

→ Gruppencontainer mit eigener Sub-Kette. Gestrichelter oranger Rahmen. elif/else-Blöcke darunter anhängbar.

Zusammenfassung der Unterschiede:
① Layer-WENN Badge oben rechts am Node — ein Textfeld für die Bedingung
② Feld-WENN Mini-Chips neben dem Feld innerhalb des Nodes — wenn/dann/sonst
③ Block-WENN Gestrichelter Container-Node mit Sub-Kette — elif/else als weitere Container
